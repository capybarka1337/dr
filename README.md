# Поздравления с днем рождения

Минималистичный сайт-поздравление на React + TypeScript. Позволяет сразу отправить готовые поздравления или создать собственную открытку с уникальным оформлением и поделиться ссылкой.

## Возможности

- Красивые готовые поздравления для мамы и папы.
- Конструктор кастомного поздравления с выбором визуальной темы.
- Генерация ссылки с текстом прямо в URL (без сервера).
- Просмотр по ссылке с адаптивным, минималистичным дизайном.
- Возможность поделиться открыткой или скопировать ссылку в один клик.

## Стек

- [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/) для разработки и сборки

## Начало работы

```bash
npm install
npm run dev
```

После запуска локально сайт будет доступен по адресу `http://localhost:5173` (порт может отличаться в зависимости от окружения).

## Скрипты

- `npm run dev` — запуск локального дев-сервера Vite.
- `npm run build` — типизация и сборка проекта в папку `dist`.
- `npm run preview` — предпросмотр собранного приложения.
